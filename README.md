# EOG Maze Control using Machine Learning

## Overview
This Human-Computer Interaction (HCI) project uses Electrooculography (EOG) signals to control a maze game through eye movements.

The system classifies eye movements and converts them into game controls.

## Features
- Eye movement classification
- Machine Learning prediction
- Maze control using EOG signals
- Supports single-file and paired-file testing

## Technologies
- Python
- Scikit-learn
- NumPy
- Pandas

## Project Structure
- `maze_project.py` - Main application
- `single file_TestScript.py` - Single file testing
- `paired file_TestScript.py` - Paired file testing
- `best_single_model.pkl` - Trained model
- `best_paired_model.pkl` - Trained model
